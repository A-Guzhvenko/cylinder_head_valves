# Cylinder Head Valves

Приложение для расчёта профиля и генерации G-кода для обработки сёдел клапанов головки цилиндров двигателя в системе LinuxCNC.

## Описание

Проект представляет собой GUI-приложение на базе GTK3 для LinuxCNC, которое позволяет:

- Рассчитывать геометрию сёдел впускных и выпускных клапанов
- Визуализировать профиль клапана и седла
- Генерировать G-код для ЧПУ обработки
- Сохранять и загружать параметры проектов
- Проверять пересечение профилей при обработке

## Требования

- **LinuxCNC** - система управления ЧПУ
- **Python 3.x**
- **GTK3** - графическая библиотека
- **NumPy** - численные вычисления
- **Pandas** - работа с табличными данными
- **Matplotlib** - визуализация графиков
- **SciPy** - научные вычисления
- **Shapely** - работа с геометрическими объектами

## Компоненты

### `ValveSeatContour`
Класс для расчёта контура седла клапана на основе параметров:
- `fd` - диаметр клапана
- `vsdtr` - диаметр верхнего края седла
- `vsa` - угол седла (основной)
- `vsw` - ширина седла
- `vsa2` - дополнительный угол седла
- `vsd` - глубина седла
- `y_offset` - смещение по оси Y
- `md` - материал/инструмент

### `HandlerClass`
Основной класс обработчика событий GUI, который управляет:
- Вводом параметров через интерфейс
- Расчётом траектории инструмента
- Генерацией G-кода
- Отображением графиков профиля
- Таблицей данных проектов

## Использование

1. **Запуск**: Приложение загружается как компонент LinuxCNC через halcmd/gladevcp

2. **Ввод параметров**:
   - Диаметр клапана
   - Углы сёдел (основной и дополнительный)
   - Ширина седла
   - Глубина обработки
   - Тип инструмента

3. **Расчёт и визуализация**:
   - Нажмите кнопку расчёта для построения профиля
   - Просмотрите результат на графике

4. **Генерация G-кода**:
   - После проверки профиля сгенерируйте управляющую программу
   - Сохраните файл `.ngc` для использования в LinuxCNC

5. **Управление проектами**:
   - Сохранение текущих параметров
   - Загрузка ранее сохраненных конфигураций
   - Добавление/удаление записей в таблице

## Параметры обработки

### Входной клапан (Intake)
- Параметры вводятся в соответствующие поля интерфейса
- Расчёт учитывает геометрию впускного канала

### Выпускной клапан (Exhaust)
- Отдельные параметры для выпускного седла
- Учёт термических расширений материала

## Генерация G-кода

Программа генерирует G-код с:
- Преамбулой (настройка инструмента, координат)
- Основным циклом обработки профиля
- Завершающей частью (возврат в исходную позицию)

## Форматы файлов

- **`.ui`** - файл интерфейса Glade (GTK)
- **`.ngc`** - файл G-кода для LinuxCNC
- **`.json`** - сохранение параметров проекта
